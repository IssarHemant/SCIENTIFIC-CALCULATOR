SCIENTIFIC CALCULATOR GUIDE

  • To calculate Scientific values, first of all click on number button then click on scientific values button. For e.g. If you want to calculate square of 2 then first press 2 after that press x2.

  • Squareroot (√) Button calculates squareroot of a number. E.g. √3=1.73205080757

  • Cuberoot (∛)Button calculates cuberoot of a number. E.g. ∛4=1.58740105197

  • x2 Button calculates square of a number. E.g. 2x2=4

  • x3 Button calculates cube of a number. E.g. 2x2x2=8

  • Pow Button returns the value which is equal to the base raised to the exponent number. For e.g. If you want to calculate power of number 10 then it will multiply 10 to 10 times, same for others number also.

  • Pi Button multiply the number with its value 3.14159265359. (π=3.14159265359). For e.g. If you calculate number 7 with Pi(π) then result is 21.9911485751.

  • ex is an exponential function with a base equal to 'e', which is known as "Euler's number". It is written as f(x) = ex, where 'e' is the Euler's number and its value is approximately 2.718.

  • Rad (Radian) Button converts the value from Degree to Radian. Formula=> (degree * Math.PI) / 180

  • Deg (Degree) Button converts the value from Radian to Degree. Formula=> Radian * (180 / Math.PI)
